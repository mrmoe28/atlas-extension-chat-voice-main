Atlas Voice Extension API Server

This is the backend API server for the Atlas Voice Chrome Extension.

Available Endpoints:

GET/POST /api/ephemeral

Returns ephemeral credentials for OpenAI Realtime API connection.

GET /api/test

Test endpoint to verify API is working and environment variables are set.

GET /api/knowledge

Returns knowledge base data (mock data for now).

POST /api/desktop

Desktop control commands (only works locally, not on Vercel).
